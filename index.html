Leaflet Relief Plugin Demo

This demo showcases the Leaflet Relief plugin with both hillshade and slope visualization modes. The plugin uses AWS Terrarium elevation tiles to create terrain visualizations.

  • Hillshade Mode: Simulates sunlight on terrain to show relief
  • Slope Mode: Colors terrain by steepness (green = flat, red = steep)
  • Interactive Controls: Adjust sun azimuth and elevation in real-time
